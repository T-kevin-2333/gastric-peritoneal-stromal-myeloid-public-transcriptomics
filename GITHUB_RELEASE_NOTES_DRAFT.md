# GitHub release notes

Tag: `v0.1.1-submission`

Title: Submission reproducibility package for gastric cancer peritoneal-context public transcriptomics

## Contents

- Complete reproducibility package supplied as split ZIP archive parts for browser-upload compatibility.
- Part-level and original-archive SHA256 checksums for reconstruction auditing.
- Non-restricted derived metadata anchors and audit outputs.
- Figure source exports and publication figure outputs.
- Formal supplementary tables and manuscript-facing reproducibility materials.

## Scope boundary

This release supports the public-data prioritisation manuscript: `Evidence-bounded public transcriptomics prioritises stromal-myeloid composition for validation in gastric cancer peritoneal contexts`.
It does not redistribute raw public datasets or large intermediate single-cell/spatial objects.
