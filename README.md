# Gastric cancer peritoneal-context public transcriptomic prioritization package

This repository hosts the public release package for an evidence-bounded public transcriptomic prioritization study of gastric cancer peritoneal contexts.

## Scope

The package supports a public-data, hypothesis-generation workflow. It prioritizes a peritoneal-context stromal-myeloid composition axis and a secondary epithelial model-response layer. It does not redistribute raw public datasets and should not be interpreted as a mechanism, clinical-deployment, or experimental-validation repository.

## Included materials

- `repository_release_package_upload_candidate_v0.1.0-submission.zip`: complete reproducibility package prepared for submission support.
- `GITHUB_RELEASE_NOTES_DRAFT.md`: release notes used for the GitHub release.
- `ZENODO_METADATA_DRAFT.json`: metadata draft for Zenodo deposition.
- `DATA_AVAILABILITY_REPLACEMENT_DRAFT.md`: manuscript data/code availability replacement text after DOI assignment.
- `REPOSITORY_ZENODO_RELEASE_CHECKLIST.md`: release and archival checklist.
- `LICENSE`: MIT License for code/scripts.
- `LICENSE_DATA_DOCS.md`: CC BY 4.0 statement for non-code derived materials.
- `CITATION.cff`: citation metadata.

## Excluded materials

Raw downloaded public datasets, unpacked expression matrices, large intermediate objects, CellOracle runtime objects, archived figure drafts, demoted boundary panels, and journal-specific cover letters are intentionally excluded. Raw public datasets should be downloaded from their original public accessions.
